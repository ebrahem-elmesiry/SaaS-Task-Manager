# 🚀 Task Management SaaS Platform

A modern SaaS-based task management system designed to help users organize work and track progress through a clean and interactive Kanban workflow.

The project focuses on building a scalable frontend architecture with real-time state management and backend integration using Supabase.

---

## 💡 Project Overview

The platform provides a workspace for managing tasks and projects in a structured and user-friendly interface. It simulates real-world SaaS behavior with authentication, role-based structure (basic), and dynamic data handling.

The system is built with a focus on frontend performance, reusable components, and clean state management patterns.

---

## 🧩 Core Features

### 📊 Dashboard

- Displays key statistics such as total tasks, completed tasks, and tasks in progress.
- Shows basic overview of user activity and workload.

---

### 📁 Projects Management

- Projects displayed in a simple card-based layout.
- Each project includes basic progress tracking and related tasks.

---

### ✅ Task Management System

- Kanban-style board with columns: Todo, In Progress, Review, Done.
- Supports drag & drop task movement between columns.
- Each task includes title and basic metadata.

---

### 👥 Team Management (Basic)

- Simple team members view with role labels.
- Displays assigned users per project.

---

### 👤 User Profile

- Displays basic user information and assigned tasks.

---

### ⚙️ Authentication & Data Handling

- User authentication using Supabase.
- Persistent session management.
- CRUD operations for tasks and projects.

---

## 🛠 Tech Stack

- Frontend: React, Next.js, TypeScript, Tailwind CSS
- Backend Integration: Supabase (Auth, Database)
- State Management: React state patterns + context
- UI: Modern reusable components architecture

---

## 🧠 Key Highlights

- Clean and scalable Kanban board implementation
- Drag & drop task management
- Real-world SaaS UI structure
- Strong focus on frontend architecture and state management
- Backend integration using Supabase for authentication and data handling
