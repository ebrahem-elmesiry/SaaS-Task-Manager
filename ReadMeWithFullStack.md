## 🚀 Task Management SaaS Platform

A modern SaaS-based task management system designed to help teams organize work, track progress, and improve productivity through a structured and interactive workflow.

This project focuses on building a scalable multi-role system (Admin, Manager, Member) with a strong emphasis on frontend architecture, state management, and backend integration using Supabase.

---

## 💡 Project Overview

The platform provides a complete workspace for managing projects, tasks, and teams in a highly interactive and user-friendly interface. It is designed as a real-world SaaS application with role-based access control and dynamic data handling.

The system includes advanced search functionality that allows users to search across tasks, projects, employees, and pages using a single unified search bar.

---

## 🧩 Core Features

### 📊 Dashboard

- Displays key statistics such as completed tasks, ongoing tasks, team size, and productivity metrics.
- Includes visual analytics using charts and stat cards.
- Shows recent activity for real-time tracking of updates.

---

### 📁 Projects Management

- Projects displayed as interactive cards.
- Each project includes progress tracking, team members, deadlines, and task distribution.
- Provides a clear overview of project status and workload.

---

### ✅ Task Management System

- Kanban-style board with four columns: Todo, In Progress, Review, Done.
- Each task includes title, description, comments, subtasks, mentions, and due dates.
- Clicking a task opens a detailed modal with full activity history and metadata.

---

### 👥 Team Management

- Team members displayed as cards with role-based access (Admin, Manager, Member).
- Shows performance metrics such as completed tasks and assigned projects.
- Includes invitation system for adding new members.

---

### 👤 User Profile

- Displays personal information, skills, activity history, and task performance.
- Allows users to receive invitations and manage their participation in projects.

---

### ⚙️ Settings & Security

- Profile and account settings management.
- Notification preferences control.
- Security settings including password updates and active sessions management.

---

### 🔔 Notification System

- Real-time notifications for task updates, invitations, and admin actions.
- Role-based alerts depending on user activity and permissions.

---

## 🛠 Tech Focus

- Frontend: React, Next.js, TypeScript, Tailwind CSS
- Backend Integration: Supabase (Auth, Database, RLS, Realtime, Storage)
- UI: shadCN, React Query, modern component architecture

---

## 🧠 Key Highlights

- Role-based system architecture (Admin / Manager / Member)
- Unified search system across the platform
- Scalable UI design with reusable components
- Strong focus on real-world SaaS patterns and UX flow
